# PDFGround

A powerful, privacy-focused offline PDF toolkit built with React, TypeScript, and Vite.

## Features

- **Merge PDF** - Combine multiple PDF files into one
- **Split PDF** - Split a PDF into multiple files
- **Compress PDF** - Reduce PDF file size
- **Convert PDF** - Convert images to PDF and vice versa
- **OCR Tool** - Extract text from images and scanned PDFs
- **Reorder Pages** - Rearrange PDF pages
- **Delete Pages** - Remove unwanted pages from PDFs
- **Extract Pages** - Extract specific pages as new PDFs
- **Export PDF** - Export PDFs in different formats
- **Password Protect** - Add password protection to PDFs
- **Batch Process** - Process multiple files at once

## Privacy First

All PDF processing happens locally in your browser. No files are ever uploaded to any server.

## Tech Stack

- React 19
- TypeScript
- Vite
- TailwindCSS
- pdf-lib
- Framer Motion

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
